Trabalhando com Listas

Listas Ordenadas

  1. Acordar
  2. Colocar as lentes
  3. Tomar café
  4. Escovar os dentes

Também é possível personalizar o início das suas listas, utilizando a tag <ol type="">:

Exemplo 1:

  1. Acordar
  2. Colocar as lentes
  3. Tomar café
  4. Escovar os dentes

Exemplo 2:

  1. Acordar
  2. Colocar as lentes
  3. Tomar café
  4. Escovar os dentes

Além disso, também é possível iniciar a lista com um número ou letra diferentes, utilizando a tag <ol start="">:

  1. Acordar
  2. Colocar as lentes
  3. Tomar café
  4. Escovar os dentes

Listas não ordenadas

  • Ovos
  • Cuscuz
  • Café

Também é possível alterar os ícones dos itens da lista utilizando a tag <ul type="">(disc, circle, square):

Exemplo 1:

  • Ovos
  • Cuscuz
  • Café

Exemplo 2:

  • Ovos
  • Cuscuz
  • Café
